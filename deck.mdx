## Automated forms

```jsx
<Components.SmartForm collection={Movies} />
```

```jsx
<Components.SmartForm 
  collection={Movies}
  documentId={this.props.documentId} />
```

## Permission system

```js
const schema = {
  name: {
    type: String,
    label: 'Name',
    optional: false,
    canRead: ['guests'],
    canCreate: ['members'],
    canUpdate: ['admins', Users.owns]
```
